 Formulaire de contact avec validation dynamique



 Description
Ce projet est un formulaire de contact interactif avec validation en temps réel.  
Il vérifie automatiquement les champs suivants à chaque frappe :

-  Nom complet (prénom + nom)
-  Numéro de téléphone (10 chiffres)
-  Adresse email (format valide)
-  Message (minimum 30 caractères)

 Technologies utilisée
- HTML5 – Structure sémantique
- CSS3 – Design moderne, flexbox, responsive
- JavaScript pur – Validation dynamique, expressions régulières, manipulation du DOM

 Fonctionnalités
- Validation en direct grâce à l’événement `onkeyup`
- Messages d’erreur clairs sous chaque champ
-  Icône verte  quand le champ est valide
-  Blocage de l’envoi si des erreurs persistent
-  Message temporaire en cas de tentative d’envoi échouée
